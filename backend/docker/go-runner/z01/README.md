# z01 package

Place the zone01 z01 package source files here before building the Docker image.

The z01 package provides functions like:
- z01.PrintRune(r rune)   — prints a single rune
- z01.PrintStr(s string)  — prints a string
- z01.PrintNbr(n int)     — prints an integer

These are used in exercises where fmt is forbidden.

Get the source from your zone01 campus Gitea or ask your admin.
